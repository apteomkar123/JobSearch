1. INDEED MCP JOB SEARCH — PARTIAL RESULTS
   The Indeed MCP tool returned limited results for environmental/sustainability
   jobs in the Raleigh/NC area. Many search queries returned 0 results.
   The 50 new jobs combine:
   - ~15 jobs with verified Indeed.com direct links (real active postings)
   - ~35 jobs from LinkedIn search URLs and company career pages
   These are real companies actively hiring for these types of roles in 2026,
   but the LinkedIn/company URLs are search-result links, not individual postings.
   Use the "Parse & Add Job" feature to replace these with specific posting links
   when you find them.

2. RESUME CHUNKS (resumes_0.json through resumes_11.json)
   The state.js includes code to load pre-built resume PDF chunks from
   /resumes_0.json through /resumes_11.json. These files are not included
   in the repo. Resumes are built in-browser when you use the dashboard.

3. THE ORIGINAL index (6).html IS PRESERVED
   The original file is kept for reference. The new index.html is the
   production file. The original can be deleted once the new version is
   verified working in production.
